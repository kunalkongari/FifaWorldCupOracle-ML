## 🧠 How the model works

### 1. ELO Ratings
Dynamic strength ratings computed over **928 World Cup matches (1930–2022)**:
- Every team starts at **ELO 1500**
- Winning earns points, losing loses them
- Goal-difference multiplier: larger wins → bigger ELO shift
- Home advantage: **+50 ELO** for the home side

### 2. Feature Engineering (9 features)
| Feature | Description |
|---|---|
| `HomeELO` / `AwayELO` | Pre-match ELO ratings |
| `ELO_diff` | HomeELO − AwayELO |
| `HomeFormScored` / `HomeFormConceded` | Avg goals scored/conceded (last 10 matches) |
| `AwayFormScored` / `AwayFormConceded` | Same for away team |
| `H2H_HomeWinRate` | Historical home win rate in this matchup |
| `H2H_DrawRate` | Historical draw rate in this matchup |

### 3. XGBoost Classifier
- 300 trees, max depth 4, learning rate 0.04
- Predicts probability of Home Win / Draw / Away Win
- **5-fold CV accuracy: 51.5%** vs 33.3% random baseline

### 4. Hybrid Prediction
```
Final probability = 70% XGBoost + 30% Poisson model
```
The Poisson component anchors extreme predictions.

### 5. Monte Carlo Simulation
- Run the full 2026 tournament **N times** (recommended: 5,000–10,000)
- Each match outcome is **sampled** from predicted probabilities
- Outputs: win%, final%, semi%, QF% per team

## 🔧 What improved vs the original project

| | Original | This project |
|---|---|---|
| Team strength | Static average goals/match | **Dynamic ELO** (updates every match) |
| Model | Poisson formula only | **XGBoost + Poisson hybrid** |
| Features | 2 | **9 engineered features** |
| Evaluation | None | **5-fold CV, confusion matrix, learning curve** |
| Simulation | One fixed bracket | **Monte Carlo: win probabilities** |
| Visualisations | None | **8 charts + interactive dashboard** |
| Code | Notebooks + checkpoints mess | **4 clean numbered notebooks** |
